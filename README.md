Python Scientific Calculator

//Project Description

This is a menu-driven scientific calculator developed using Python. The calculator performs arithmetic and scientific operations using functions and exception handling.

//Features

- Addition
- Subtraction
- Multiplication
- Division
- Power Calculation
- Square Root
- Factorial
- Error Handling using try-except

// Technologies Used
- Python
- Math Module

// working process
1. Install Python
2. Download the project files
3. Open terminal or command prompt
4. Run the command: python calculator.py


//Project Structure
- calculator.py → Main source code
- README.md → Project documentation


// Output
The calculator displays a menu where users can select operations repeatedly until exit.
